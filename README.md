# ReactDataGrid

## Simple Regex Filter and Sort with React.js and Immutable-js
## Existing Features:

- sortable columns
- multiple column filter
- large datasets
- modular 

## Coming Soon:
- scrolling table
- cell format
- pagination
- cell inline edit
- insert & delete row
